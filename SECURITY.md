# Security Policy

xiboplayer is distributed AS-IS under AGPL-3.0-or-later. This is an
open-source project — **no warranties, no SLAs, no guarantees.**

## Reporting a vulnerability

Please report privately to **security@xiboplayer.org**
(PGP fingerprint `991E 74C3 A033 673F 4FCF 25B8 B7D2 5A81 02F6 3D6A`,
public key on [keys.openpgp.org](https://keys.openpgp.org/vks/v1/by-email/security@xiboplayer.org))
or via [GitHub Security Advisories](../../security/advisories/new).

We'll respond as maintainer time allows. We make no commitment to a
specific response time.
